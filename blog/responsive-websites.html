Building Responsive Websites with Modern CSS

← Back to Blog

When I built this website from scratch, my goal was to master responsiveness and layout control without relying on external frameworks. I wanted to understand how design translates into code.

I started with Flexbox for one-dimensional alignment — perfect for my navbar and content sections — and then layered in CSS Grid for more complex two-dimensional layouts on the portfolio page.

The challenge was maintaining design consistency while ensuring that every page, from blog posts to contact forms, looked clean on both mobile and desktop.

The result is a layout that's fully responsive and fast-loading, using only modern CSS and semantic HTML tags. The mint-green theme reflects simplicity and calm, while the minimalist typography keeps it personal and professional.

This site is more than just a portfolio — it's a reflection of my approach to development:

"Start simple. Iterate. Optimize."

Design Philosophy

  • Mobile-first approach - Design for small screens first, then enhance for larger ones
  • Semantic HTML - Use proper tags for accessibility and SEO
  • CSS Grid and Flexbox - Modern layout techniques without frameworks
  • Consistent spacing - Use CSS variables for maintainable design systems

Technical Implementation

The website uses a combination of Flexbox for component-level layouts and CSS Grid for page-level layouts. All spacing and colors are defined using CSS custom properties for easy maintenance.

Key CSS Features Used:

  • CSS Custom Properties for consistent theming
  • Flexbox for navigation and component layouts
  • CSS Grid for portfolio and blog layouts
  • Media queries for responsive breakpoints
  • CSS animations for smooth interactions

Performance Considerations

By avoiding external frameworks and keeping the CSS lean, the site loads quickly and performs well across all devices. The use of semantic HTML also improves accessibility and SEO.

Technical Stack
